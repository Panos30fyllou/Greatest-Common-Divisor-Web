Εύρεση Μέγιστου Κοινού Διαιρέτη

Να βρεθεί ο Μέγιστος Κοινός Διαιρέτης (ΜΚΔ) δύο θετικών ακεραίων αριθμών x και y.


Ο Πρώτος αλγόριθμος που θα παρουσιάσουμε είναι ο εξής:


Αλγόριθμος ΜΚΔ
Δεδομένα // x,y //
Αν x < y τότε
z ← x
αλλιώς
z ← y
Τέλος_αν
Όσο (x mod z ≠ 0) ή (y mod z ≠ 0) επανάλαβε
z ← z-1
Τέλος_επανάληψης
Αποτελέσματα // z //

Αλγόριθμος Ευκλείδης

z ← y
Όσο z ≠ 0 επανάλαβε
z ← x mod y
x ← y
y ← z

Εκτελώντας τους 2 αλγορίθμους για διάφορες τιμές x, y, διαπιστώνουμε πως ο αλγόριθμος του Ευκλείδη είναι ταχύτερος, συνεπώς και ο επικρατέστερος.
Πατήστε F5 για επανάληψη της διαδικασίας.
